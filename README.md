# TripNest Travels ✈️🌍

TripNest Travels is a responsive travel booking website designed to provide users with an easy and smooth experience for exploring destinations, planning trips, and making bookings online.

## Features 🚀

* Responsive modern travel website design
* Attractive homepage with travel destination sections
* Booking form for trip reservations
* Dark mode support
* Clean and user-friendly interface
* Fast navigation between sections

## Technologies Used 💻

* HTML
* CSS
* JavaScript
* Tailwind CSS

## Project Structure 📂

* `index.html` – Main homepage
* `styles.css` – Custom styling
* `script.js` – Website functionality
* `booking-form.html` – Booking form page
* `booking-form-styles.css` – Booking form styling
* `booking-form-script.js` – Booking form logic

## How to Run the Project ▶️

1. Download or clone the repository
2. Open the project folder in VS Code
3. Run `index.html` in browser using Live Server
